# 🛒 Amazon Clone (HTML & CSS)

This is my **first front-end web development project** where I built a clone of the Amazon homepage using only **HTML and CSS**.

Although I am currently unable to upload the project to GitHub, I wanted to properly document it and share the details of my work.

---

## 📌 Project Overview

The goal of this project was to:

- Practice HTML structure
- Improve CSS styling skills
- Understand layout techniques (Flexbox, positioning, etc.)
- Recreate a real-world website design
- Build confidence in front-end development

I chose to recreate the homepage of Amazon as it contains a wide range of UI components like navigation bars, product sections, banners, and footers.

---

## 🚀 Features

- ✔ Navigation Bar with logo and search bar  
- ✔ Hero Section (Banner Area)  
- ✔ Product Categories Section  
- ✔ Grid Layout for Products  
- ✔ Footer Section  
- ✔ Hover Effects  
- ✔ Basic Responsive Layout  

---

## 🛠 Technologies Used
